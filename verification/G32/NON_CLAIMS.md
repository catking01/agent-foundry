# G32 Non-Claims

G32 public research demo release does not claim:

- production hosting, deployment, or SLA
- a real-world optimal organization structure
- a real-world best AI organization policy
- validation of multi-agent organization theory against real organizations
- real AI supervisor, lead, or worker execution in the demo
- Runtime Lab integration or validation
- Ollama or live LLM participation in policy search results
- live external API calls from the browser UI
- GitHub Pages deployment completion
- exhaustive organization-design search
- that deterministic G30 simulator results generalize to real organizations
- that the Pareto frontier represents a universal optimum
- mutation, dispatch, or game-state-changing capability in the Research UI

Correct claim:

Agent Foundry is a public research demo: a deterministic multi-agent organization simulator with read-only study dashboards and policy search results, validated at commit `6c9c448`.
