# Non-Claims

Agent Foundry does NOT claim:

- Real AI employees or autonomous agents
- Real LLM workers, Codex agents, or Claude integration
- Real multi-agent research capability
- Validation of Runtime Lab or any AI company theory
- Production-grade evidence governance or security audit
- Real-time observability or production SRE
- Proven organizational optimization
- General player fun or mass-market game readiness

Agent Foundry IS:

- A deterministic simulated AI-company organization game
- An exploration sandbox for multi-agent pipelines, validation, audit, and trust
- A machine-readable balance experiment with replay integrity
- An optional local LLM shadow evaluation demo (never authoritative)
