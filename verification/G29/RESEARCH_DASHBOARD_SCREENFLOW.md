# G29 Research Dashboard Screenflow

## Entry

```text
App header/tab bar -> Research
```

## Flow

1. Open the main Agent Foundry app.
2. Select the `Research` tab.
3. Read the `Deterministic study results` overview.
4. Compare G27 flat and hierarchical metrics.
5. Review G28 intervention ranking.
6. Inspect delta metrics and complexity highlights.
7. Read risk semantics and non-claims before interpreting results.

## Interaction Model

The dashboard is read-only. It contains no mutation buttons, run controls, dispatch hooks, or editable inputs.
